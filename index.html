Pattern

SVG Pattern Element

Scalable Vector Graphic (SVGs) can be used for web development and data visualization. It is mainly used to create vector based images. SVG draws lines at different coordinates on the screen to create a image. A powerful SVG element is pattern!

The pattern element defines a graphics object which can be redrawn at repeatedx and y coordinate intervals ("tiled") to cover an area.

Most of the time a pattern is filling a shape. To accomplish is in the SVG element with give the shape a style property fill that is referencing the pattern defined above.
